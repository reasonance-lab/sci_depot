Usage Examples

Inline SVG (Recommended for animations)

<!-- Copy SVG content directly into your HTML -->
<div class="flask-container">
    <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 100 140">
        <!-- SVG content here -->
    </svg>
</div>

Image Tag

<img src="svg-library/flasks/erlenmeyer-flask-empty.svg"
     alt="Erlenmeyer Flask"
     width="100"
     height="140">

CSS Background

.beaker-icon {
    background-image: url('svg-library/beakers/beaker-empty.svg');
    background-size: contain;
    background-repeat: no-repeat;
    width: 80px;
    height: 100px;
}

JavaScript Dynamic Loading

// Load and manipulate SVG
fetch('svg-library/flasks/erlenmeyer-flask-empty.svg')
    .then(response => response.text())
    .then(svgText => {
        const container = document.getElementById('flask-container');
        container.innerHTML = svgText;

        // Now you can manipulate the SVG
        const liquid = container.querySelector('#liquid');
        liquid.style.fill = '#ff6b6b'; // Change liquid color
    });

Animation Example (CSS)

/* Animate liquid level */
@keyframes fillUp {
    from { transform: scaleY(0); }
    to { transform: scaleY(1); }
}

.liquid-layer {
    transform-origin: bottom;
    animation: fillUp 2s ease-out forwards;
}

/* Animate bubbling */
@keyframes bubble {
    0%, 100% { transform: translateY(0); opacity: 0; }
    50% { transform: translateY(-20px); opacity: 1; }
}

.bubble {
    animation: bubble 1.5s infinite;
}

Customizing Colors

Each SVG uses CSS gradients and colors that can be modified:

  • #7ec8e3 - Default liquid blue
  • #e88080 - Cell culture medium pink/red
  • #f5c556 - Yellow liquid
  • #e87090 - Titration endpoint pink
  • #6ba3be - Glass outline color
